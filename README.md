# ucvme

Make sure you have Node.js version 18.0 or higher on your machine.

## Project Setup

```sh
git clone https://github.com/ontingleung/ucvme
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
